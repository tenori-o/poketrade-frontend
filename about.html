About


This calculator uses two external APIs for analyzes pokemon tradings and show info about them. Here is the Swagger for the Poke Trade API and the docs for the Poke Api.

Is a full RESTful API linked to an extensive database detailing everything about the Pokémon main game series.

They have an awesome documentation about how to use it. It takes minutes to get started.

The API analyzes whether the pokemon trading between players can be considered fair. A trade consists of two players offering between 1 and 6 pokemons on each side. Anyone is valid.

Ex: Player 1 tries to exchange 3 pokemons with 2 pokemons from player 2.
If the difference of the lowest value "base experience" is 15% less than the other, the exchange is unfair.
